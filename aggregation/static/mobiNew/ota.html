How is this possible?

MobiWiSD's technology examines hardware characteristics of a device, which enables multiple factors of authentication. In order for a device to connect to an organizations internal network and access data, multiple levels of authentication occur. This basic authentication is leveraged by our patent pending technology, where we are able to distinguish with 100% accuracy between different types of devices based on hardware characteristics of the device(s). Because we can establish this low-level autnetication, we can gaurentee that each device will have unique access to data. This unique acess is what enables administrators to control access to data at the lowest level and in a granular fashion. We call this the "base authentication."
